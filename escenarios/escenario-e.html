MN/BOL
    ⬡ Escenario E

    Umbrales Críticos
    de Abastecimiento

    Cálculo de puntos de equilibrio mediante raíces de ecuaciones no lineales: dónde el costo supera el ingreso, la tasa de reposición crítica de carburante y el umbral de opinion social.

    Contexto del Problema

    MODELO
    Situación Bolivia: ¿En qué día el gasto de una familia supera su ingreso mensual? ¿A qué tasa de reposición el combustible se equilibra? ¿Cuándo el descontento social cruza el umbral crítico? Estas preguntas se responden encontrando raíces de ecuaciones no lineales.

    Selección del Problema

    FORMULARIO
    Problema: f(t) = Gasto_acumulado(t) − Ingreso_disp(t) = 0
    Encontrar el día t* donde el gasto acumulado iguala el presupuesto familiar.
    f(t) = P₀·e^(r·t)·q·t − I = 0
    donde P₀=precio inicial, r=tasa, q=cantidad/día, I=ingreso mensual
    Problema: g(Q) = R₀ + Q·T − D₀·T = 0
    Encontrar la tasa de entrada Q* que equilibra el consumo D₀ durante T días.
    g(Q) = (Q − D₀)·T + R₀ − R_crit = 0
    Q* = tasa crítica de reposición para no vaciar las reservas
    Problema: h(x) = descontento − umbral_estabilidad = 0
    Encontrar el nivel de malestar social x* donde el sistema cambia de estable a masificado.
    h(x) = a·x³ − b·x² + c·x − d = 0
    Polinomio que modela la función de tensión social vs. respuesta institucional

    Preguntas que Responde el Modelado

    RESPUESTAS

    1. En que dia el gasto iguala al ingreso?

    Se resuelve f(t)=Gasto(t)-Ingreso=0. La raiz t* marca el dia en que la familia deja de cubrir la canasta con su presupuesto disponible.

    2. Cual es la tasa minima de reposicion?

    Se resuelve g(Q)=0 para encontrar Q*. Si la entrada diaria es menor que esa raiz, la reserva cae por debajo del nivel critico.

    3. Cuando el descontento cruza el umbral?

    Se resuelve h(x)=0. La raiz indica el nivel de tension donde el sistema pasa de malestar contenido a riesgo de movilizacion masiva.

    4. Que metodo de raices es adecuado?

    Biseccion es robusta si hay cambio de signo; Newton-Raphson es rapido con buena condicion inicial; Secante evita derivadas y funciona como punto medio practico.

    Métodos de Búsqueda de Raíces

    TEORÍA
    x_mid = (a+b)/2
    Si f(a)·f(x_mid) < 0 → b=x_mid, else a=x_mid

    Bisección divide el intervalo a la mitad iterativamente. Convergencia garantizada si existe raíz en [a,b]. Convergencia lineal — O(log₂((b-a)/ε)) iteraciones. Robusto pero lento.

    x_{k+1} = x_k − f(x_k)/f'(x_k)

    Newton-Raphson usa la tangente para converger cuadráticamente. Muy rápido cerca de la raíz. Requiere f'(x) — se calcula numéricamente con diferencias finitas. Puede divergir si la condición inicial está lejos de la raíz o f'≈0.

    x_{k+1} = x_k − f(x_k)·(x_k − x_{k-1})/(f(x_k) − f(x_{k-1}))

    La secante evita calcular f'. Usa dos puntos anteriores para aproximar la derivada. Convergencia superlineal (≈1.618). Balance ideal entre bisección y Newton-Raphson para umbrales de abastecimiento.
